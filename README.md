### Capstone Project - Data Analytics with SQL (TeoMeWhy Course)

---

🇧🇷 **Descrição do Projeto**
Este repositório contém o projeto prático final desenvolvido por mim durante o curso de SQL ministrado pelo Teo Calvo (TeoMeWhy). O objetivo principal foi aplicar técnicas avançadas de manipulação, ETL e análise de dados utilizando SQLite para resolver problemas reais de negócios.

🇺🇸 **Project Description**
This repository contains the final practical project developed during the SQL course taught by Teo Calvo (TeoMeWhy). The main objective was to apply advanced data manipulation, ETL, and data analysis techniques using SQLite to solve real-world business problems.

---

## 🛠️ Tecnologias / Technologies
* **Language:** SQL
* **SGBD:** SQLite
* **Editor:** VS Code / Terminal macOS
* **Versionamento / Version Control:** Git & GitHub

## 📈 Conceitos de SQL Aplicados / SQL Concepts Applied
* **ETL (Extract, Transform, Load):** Estruturação e carga de dados brutos para tabelas analíticas.
* **Subqueries & CTEs (Common Table Expressions):** Para estruturação de consultas complexas e legibilidade do código.
* **Window Functions:** Aplicadas para análises temporais e ordenação de dados.
* **Funções de Agregação e Joins:** (`LEFT/INNER JOIN`, `GROUP BY`, `HAVING`) para consolidação de métricas.
